# 📖 Descripción

CleanBot-RCR es un **prototipo funcional** desarrollado como proyecto de título para validar una solución tecnológica orientada a la automatización de la limpieza de señalética en faenas mineras.

La solución integra una plataforma web desarrollada en **Node.js**, una base de datos **MySQL**, una **ESP32-CAM** para identificación mediante códigos QR y un sistema robótico controlado por **Arduino**, permitiendo demostrar la factibilidad técnica de automatizar este proceso y reducir la exposición de operadores a tareas de riesgo.

# ⚠️ Problema

La limpieza de señalética en entornos mineros es una tarea periódica que puede exponer a los operadores a riesgos asociados al tránsito de maquinaria pesada, condiciones ambientales adversas y trabajos en zonas de difícil acceso.

Además, el proceso manual incrementa los tiempos de mantenimiento y requiere una mayor intervención humana.

Frente a este escenario surgió la necesidad de diseñar una solución tecnológica que permitiera automatizar este proceso y disminuir la exposición del personal.

---

# 💡 Solución

Se desarrolló un prototipo funcional que integra tecnologías de automatización, IoT y desarrollo web para demostrar la viabilidad de automatizar la limpieza de señalética.

El sistema está compuesto por:

- Plataforma web desarrollada en Node.js y Express.
- Base de datos MySQL.
- ESP32-CAM para lectura de códigos QR.
- Arduino Mega para el control del brazo robótico.
- Motores paso a paso y actuadores.
- Sistema de limpieza mediante hidrolavadora.

La plataforma coordina la comunicación entre todos los componentes permitiendo seleccionar una señal, verificar su identificación y ejecutar el proceso de limpieza desde una interfaz web.
